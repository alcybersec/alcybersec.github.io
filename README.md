# AlCyberSec Personal Website

![Work In Progress](https://img.shields.io/badge/status-WIP-blue.svg)

This is the repository for the AlCyberSec personal website, showcasing cybersecurity projects, privacy advocacy articles, and programming skills.

## Overview

The website includes:

- **Home Page**: Introduction and summary of expertise.
- **Blog**: Articles on privacy, cybersecurity, and related topics.
- **Skills**: Detailed list of skills and tools.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [HTML5 UP Landed Template](https://html5up.net/landed)
